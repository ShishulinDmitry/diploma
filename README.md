## Рекомендации по компиляции и запуску

### Компиляция `etalon.c` (без оптимизаций)

Для базовой версии без флагов оптимизации рекомендуется использовать стандартную компиляцию с помощью `gcc`:

```bash
gcc etalon.c -o etalon
```

---

### Компиляция `etalon_opt.c` (оптимизированная версия с OpenMP)

Для оптимизированной версии рекомендуется включить высокий уровень оптимизации, генерацию кода под текущую архитектуру процессора и поддержку OpenMP:

```bash
gcc etalon_opt.c -O3 -march=native -fopenmp -o etalon_opt
```

---

### Компиляция CUDA-версии `cuda_version.cu`

Для CUDA-реализации используется компилятор `nvcc`. Рекомендуется включить оптимизацию `-O3` и указать архитектуру GPU.

Пример для архитектуры **sm_89**:

```bash
nvcc cuda_version.cu -O3 -arch=sm_89 -o cuda_version
```

При необходимости замените `sm_89` на архитектуру вашей видеокарты.

---

### Запуск и компиляция `sage.py`

Файл `sage.py` предназначен для выполнения в онлайн-среде SageMath.

1. Перейдите на сайт:  
   https://sagecell.sagemath.org/

2. Вставьте содержимое файла `sage.py` в окно редактора.

3. Нажмите кнопку **Evaluate** для выполнения кода.

Локальная компиляция не требуется — все вычисления выполняются на стороне сервера SageMath.
